## 一、蓝网科技超声RIS

### 1. 产品定位
蓝网科技是国内专注于超声信息化的厂商，其超声RIS系统主要面向二级及以上医院的超声科，提供完整的超声检查工作流和报告管理。

### 2. 整体架构
- **架构类型**：B/S架构，基于Web的超声科室管理系统
- **部署方式**：本地化部署，支持私有云
- **数据库**：支持Oracle/SQL Server/MySQL
- **标准支持**：DICOM MWL、HL7 v2.x、DICOM 3.0

### 3. 核心功能模块
- **检查预约**：超声检查自助预约，支持叫号分诊
- **工作列表**：DICOM MWL集成，实时检查进度跟踪
- **报告书写**：结构化报告模板，支持语音输入
- **图像管理**：超声影像采集、存储、调阅，支持超声特有动态图像
- **PACS集成**：与各类PACS系统对接，支持超声图像归档
- **统计分析**：检查量统计、工作量统计、收入分析
- **质量管理**：报告质控、危急值管理

### 4. 特色功能
- 超声科专用分诊叫号系统，与检查流程深度集成
- 支持多种探头类型的检查分类管理
- 超声图像与报告一体化展示
- 超声特有"检查部位-检查方法"标准化字典

### 5. 技术亮点
- 支持超声设备DICOM SCU/SCP双模式连接
- 动态图像（Clip）存储与回放
- 结构化模板引擎，支持联想输入

## 二、东软医疗超声RIS（Neusoft PACS/RIS）

### 1. 产品定位
东软医疗的PACS/RIS是其智慧医学影像信息系统的核心组成部分，面向大型三级医院，支持放射、超声、内窥镜、病理、电生理、核医学、介入等多科室。

### 2. 整体架构
- **架构类型**：B/S架构，企业级云化架构
- **部署方式**：支持本地化、私有云、公有云混合部署
- **数据库**：支持大型数据库集群（Oracle RAC）
- **标准支持**：完全符合DICOM 3.0、HL7 v2.x/IHE规范
- **技术栈**：负载均衡、集群部署，支持全院影像数据集中存储

### 3. 核心功能模块（官网提取）

| 模块 | 功能描述 |
|------|---------|
| **影像存储与调阅** | 影像数据永久安全管理，负载均衡面向全院提供调阅及后处理服务 |
| **专科诊断工作站** | 支持放射、超声、内窥镜、病理、电生理、核医学、介入等多科室 |
| **业务流程信息化** | 检查预约、登记、检查、执行、报告全流程闭环 |
| **智能辅助诊断** | AI辅助诊断集成（基于疾病诊断知识库） |
| **结构化报告** | 基于疾病诊断知识库的结构化报告，支持模板定制 |
| **危急值管理** | 检查闭环和危急值闭环完全管理 |
| **临床系统交互** | 与HIS/EMR/电子病历系统深度集成 |
| **绩效管理** | 全面的业务、质量、绩效管理，多维度绩效考核 |
| **质控管理** | 切实加强影像质量安全管控 |
| **教科研支持** | 支撑影像教科研发展 |

### 4. 特色功能
- **多科室统一平台**：超声+放射+病理等多模态统一管理
- **检查闭环+危急值闭环**：双闭环设计，完全符合互联互通、电子病历等评级要求
- **N-VNA智慧影像数据平台**：统一影像数据管理，支持跨科室调阅
- **AI辅助诊断集成**：基于疾病诊断知识库的结构化报告
- **全面评级解决方案**：提供互联互通、电子病历等完整评级支持
- **负载均衡技术**：保障全院级大规模并发访问
- **智慧物联+AI+大数据**：新一代信息技术整合

### 5. 技术亮点
- DICOM、HL7标准及IHE规范完全兼容
- 负载均衡集群部署
- 影像数据永久安全管理机制
- 智慧物联（IoT）集成
- 人工智能辅助诊断
- 大数据分析支持
- 云平台架构支持

## 三、联影医疗超声RIS（UIHPACS）

### 1. 产品定位
联影医疗是国内高端医学影像设备供应商，其信息管理系统（UIHPACS）面向大型三甲医院，提供从设备到信息化整体解决方案。

### 2. 整体架构
- **架构类型**：B/S架构，云计算架构
- **部署方式**：支持院内私有云、区域云平台部署
- **数据库**：大型分布式数据库
- **标准支持**：DICOM 3.0、HL7 v2.x、IHE、DICOM SR结构化报告

### 3. 核心功能模块
- **检查工作流**：DICOM MWL集成，自动获取检查申请
- **报告管理**：结构化报告书写，支持图文一体化报告
- **影像管理**：支持超声动态图像（DICOM顾型）存储
- **PACS集成**：与联影CT/MR/XR等设备无缝对接
- **设备管理**：超声设备状态监控、预约管理
- **统计分析**：科室工作量、设备使用率统计
- **危急值推送**：与临床系统实时联动
- **区域互联**：支持医联体/区域影像数据中心

### 4. 特色功能
- **高端设备配套**：与联影超声设备深度集成
- **云PACS架构**：支持多院区、区域影像数据中心
- **结构化报告**：DICOM SR兼容的结构化报告
- **统一认证**：与医院一卡通/人脸识别系统集成
- **AI辅助诊断**：与联影AI平台（uAI）无缝集成

### 5. 技术亮点
- 云计算+分布式存储架构
- DICOM SR结构化报告
- 设备与信息系统深度融合
- 多院区统一管理
- 区域影像数据共享

## 四、开立医疗超声RIS（Sonoskand/开立）

### 1. 产品定位
开立医疗（Sonoskand）是以超声设备为核心的公司，其超声信息系统面向超声科，提供从设备到信息化的完整生态链。

### 2. 整体架构
- **架构类型**：B/S架构，模块化设计
- **部署方式**：本地化部署为主
- **数据库**：SQL Server/MySQL
- **标准支持**：DICOM 3.0、HL7 v2.x

### 3. 核心功能模块
- **超声工作列表**：支持DICOM MWL，实时检查状态
- **报告书写**：结构化报告模板，支持图像标注
- **图像管理**：支持动态图像（DICOM Clip）采集存储
- **预约管理**：检查预约、分时段预约
- **统计分析**：检查量统计、医生工作量统计
- **质量控制**：报告质控评分、修改痕迹保留
- **词库管理**：超声专业词库，支持联想输入
- **会诊管理**：超声远程会诊、线上阅片

### 4. 特色功能
- **设备原生集成**：与开立超声设备（S50/S60等）无缝对接
- **超声动态图像**：专利压缩算法，支持长时间动态图像存储
- **专科报告模板**：产科、浅表、心脏、血管、腹部等专科模板
- **云端升级**：设备系统远程升级维护
- **远程诊断**：远程超声会诊解决方案

### 5. 技术亮点
- 超声设备软硬件一体化设计
- 动态图像压缩存储技术
- 专科报告模板引擎
- 远程诊断技术支持

# G003 超声RIS系统 - 功能规划
> 版本: v0.6.0 | 更新: 2026-05-01 | 状态: 开发中

---

## 项目概述

**G003超声RIS系统** 是智慧超声影像信息管理系统，以联影医疗、东软医疗、蓝网科技、开立医疗、岱嘉医学等头部竞品为标杆，目标实现**功能最全、AI最强、数据最丰富**的国产超声RIS系统。

**技术栈**: React 18 + TypeScript + Vite + Recharts + Lucide Icons
